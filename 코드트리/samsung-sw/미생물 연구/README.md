## 🦠 미생물 연구

## 📄 문제

| 항목           | 내용      |
| -------------- | --------- |
| 난이도         | 삼성 기출 |
| 평균 풀이 시간 | 180분     |
| 정답률         | 41%       |
| 총 제출        | 4,250회   |

미생물 연구자 Sam은 논문을 작성하기 위해 실험을 계획했습니다.

실험은 $N×N$ 크기의 정사각형 형태의 배양 용기에서 진행됩니다. 이 배양 용기에는 좌표 평면이 그려져 있는데, 좌측 하단의 좌표는 $(0, \ 0)$ 가장 우측 상단의 좌표는 $(N, \ N)$으로 표현됩니다.

Sam은 배양 용기에서 총 $Q$번의 실험을 진행하며 각 실험의 결과를 기록합니다. 각각의 실험은 다음과 같은 순서로 진행됩니다.

---

## 🧬 미생물 투입

좌측 하단 좌표가 $(r_1, \ c_1)$이고 우측 상단 좌표가 $(r_2, \ c_2)$인 직사각형 영역에 한 무리의 미생물을 투입합니다.

- 영역 내에 다른 미생물이 존재하면 → 새 미생물이 기존 미생물을 모두 제거
- 즉, 해당 영역은 새 미생물로 완전히 덮임

또한,

- 기존 미생물 $A$가 제거되면서 영역이 **둘 이상으로 분리되면**
- 해당 미생물 $A$는 **완전히 소멸**

---

## 🔄 배양 용기 이동

모든 미생물을 새로운 배양 용기로 이동합니다.

다음 과정을 **기존 용기가 빌 때까지 반복**합니다.

1. 가장 넓은 영역을 가진 미생물 선택
2. 넓이가 같으면 → 먼저 투입된 미생물 선택
3. 기존 형태 그대로 유지하며 배치
4. 조건
   - 용기 범위 내
   - 다른 미생물과 겹치지 않음
5. 배치 기준
   - $x$좌표 최소
   - $x$ 동일 시 $y$좌표 최소
6. 배치 불가능하면 → 해당 미생물 제거

---

## 📊 실험 결과 기록

인접 조건:

- 상하좌우로 맞닿은 경우 인접

계산 방식:

- 모든 인접한 쌍 $(A, \ B)$에 대해  
  → `넓이(A) × 넓이(B)`

주의:

- 여러 면에서 붙어도 **1번만 계산**

최종 결과:

- 모든 쌍의 값을 더한 값

## 📝 예제 설명

### 1️⃣ 첫 번째 실험

- 하나의 미생물만 존재  
  → 결과: 0

---

### 2️⃣ 두 번째 실험

- 기존 미생물 일부 제거
- 더 넓은 미생물 먼저 배치
- 인접 발생

→ 계산:  
`15 × 11 = 165`

---

### 3️⃣ 세 번째 실험

- 기존 미생물 분리 발생  
  → 해당 미생물 제거

→ 계산:  
`9 × 12 = 108`

---

### 4️⃣ 네 번째 실험

- 3개의 미생물 존재
- 인접 쌍 2개 발생

→ 계산:  
`25 × 6 + 25 × 7 = 325`

## 풀이 핵심

- 상태 관리는 `int[][] board`로
- 미생물들끼리 겹쳐서 쪼개지는 판정은 BFS 활용
- 2개 이상으로 쪼개진 미생물은 바로 제거
- 정렬 기준에 맞춰 새로 보드에 배치
- 모양 유지는 상대 좌표 이용
- adj 행렬을 통해 인접한 미생물 쌍 기록
