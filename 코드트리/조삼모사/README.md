## 📝 문제 설명

$n$ 개의 일을 두 그룹(아침, 저녁)으로 나누어 각각 $\frac{n}{2}$ 개씩 배치합니다.

각 일 $i, j$ 사이에는 함께 수행할 때의 업무 강도 $P_{ij}$ 가 존재합니다.

각 그룹의 총 업무 강도는 다음과 같이 계산됩니다.

- 같은 그룹 내 모든 쌍 $(i, j)$ 에 대해 $P_{ij}$ + $P_{ji}$ 의 합

두 그룹(아침, 저녁)의 업무 강도 차이를 최소로 만드는 값을 구해야 합니다.

## 💡 풀이 핵심

아이디어: 조합 + 백트래킹

1. boolean[] selected로 작업 분리
   - selected[i] = true 이면 아침 작업
   - selected[i] = false 이면 저녁 작업

2. 조합으로만 뽑는다
   아침에 (1, 2, 3) 뽑고 저녁에 (4, 5, 6)이면,
   나중 가서 아침에 (4, 5, 6)을 뽑는 경우는 사실 같은 분할임.

3. 점수 계산은 $i < j$만
   - 한 팀 점수 계산 시 $i < j$ 에 대해 $P_{ij}$ + $P_{ji}$
   - 중복 계산 제거

4. 시간 복잡도
   조합 개수: 대략 $\binom{n}{\frac{n}{2}}$
   각 조합마다 점수 계산: $O(N^2)$
   전체는 대략 $O(\binom{n}{\frac{n}{2}} * N^2)$
