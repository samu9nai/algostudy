## 📝 문제 설명

코드트리는 회사 확장을 위해 택배 사업을 시작합니다. 알고리즘 회사의 이점을 이용해서 완벽한 택배 하차 알고리즘을 만들었습니다.

택배는 $N × N$ 크기의 정사각형 격자에서 진행됩니다.

---

### 1️⃣ 택배 투입

- 각 택배는 직사각형 형태이며 다음 정보로 주어집니다.
  - 시작 열: $c$
  - 가로 길이: $w$
  - 세로 길이: $h$
  - 택배 번호: $k$
- 택배는 투입 후 중력에 의해 아래로 떨어집니다.
- 바닥에 닿거나 다른 택배 위에 닿으면 멈춥니다.
- 모든 택배는 공간을 벗어나지 않음이 보장됩니다.

---

### 2️⃣ 좌측 하차

- 쌓인 택배 중 왼쪽으로 이동했을 때 다른 택배와 부딪히지 않고 빠져나갈 수 있는 택배를 선택합니다.
- 여러 개일 경우, 택배 번호($k$)가 작은 것을 먼저 하차합니다.
- 하차 후 남아있는 택배들 중 떨어질 수 있는 것들은 다시 떨어집니다.

---

### 3️⃣ 우측 하차

- 좌측 하차와 동일한 방식으로 오른쪽에서도 진행합니다.

---

### 🔁 반복

- 좌측 하차 → 우측 하차 과정을 반복합니다.
- 모든 택배가 제거될 때까지 수행합니다.

---

## 🎯 출력

- 택배가 하차되는 순서를 번호($k$) 기준으로 순서대로 출력합니다.
